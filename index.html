How This Tool Works

Built for WFM analysts, not data scientists.

  • Early in the shift the tool trusts the original forecast more than the live run rate — because there isn't enough data yet to rely on it.
  • As the shift progresses it gradually shifts weight toward real data, so your revised forecast becomes more and more reliable.
  • Variance is evaluated using a simple ±5% tolerance rule. Anything over 10% is flagged as Risk.
  • The tool separates whether changes are caused by volume (more or fewer contacts) or AHT (calls taking longer or shorter), so you know where to act.
  • FTE impact is calculated using your shrinkage and occupancy targets — the number you see is what you'd need to add or remove from the floor right now.
